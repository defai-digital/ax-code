# Quickstart

## Fastest path

1. Install [AX Code](https://ax-code.ai).
2. Install AX Code Desktop for macOS or Windows (see [Install](/install/)).
3. Verify `ax-code --version` works.
4. Launch **AX Code** from Applications, the Start Menu, or the extracted portable folder.
5. Add a project folder.
6. Connect a hosted provider, or on an eligible Apple Silicon Mac, connect AX Engine as a local provider.

You should see the AX Code Desktop session list. If it loads, you're up and running.

Use a strong UI password if you expose the instance through a reverse proxy or bind it outside localhost.

If the page doesn't load, check [Troubleshooting](/troubleshooting/).

## Which app should I use?

- use **desktop** for day-to-day work on macOS and Windows
- use **AX Engine** only on supported Apple Silicon Macs
- use hosted providers or a remote supported Mac for Windows Desktop workflows that need AX Engine
