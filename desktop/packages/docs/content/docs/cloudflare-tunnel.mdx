# Cloudflare Tunnel

Cloudflare and other public tunnels are disabled by AX Code Desktop's local-only policy. The legacy `tunnel stop` command remains available only to terminate a tunnel created by an older release. AX Code Desktop refuses to start while a recorded legacy tunnel is still active.

```bash
ax-code-desktop tunnel stop
```

Use AX Code Desktop directly on the office or offline workstation that owns the code. See [Local-only Security](/security/) for the enforced network boundary.
