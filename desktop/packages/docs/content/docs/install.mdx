# Install

There are two supported Desktop install paths:

- macOS Apple Silicon app through Homebrew or DMG
- Windows x64 app through installer or portable ZIP

Desktop source code lives in the AX Code monorepo under `desktop/`, and Desktop
release artifacts are published from the main
[AX Code GitHub releases page](https://github.com/defai-digital/ax-code/releases).
The Homebrew cask tap remains separate because it is a distribution index.

## Runtime Requirement

AX Code Desktop needs the local AX Code CLI/runtime for coding sessions. The
root [AX Code README](https://github.com/defai-digital/ax-code#readme) shows
the paired CLI and Desktop install commands. After installation, verify the CLI:

```bash
ax-code --version
```

If `ax-code` is not found, install the AX Code CLI before starting Desktop
coding sessions.

## macOS

Homebrew is the recommended macOS path. Install the CLI runtime and Desktop app together on a new machine:

```bash
brew tap defai-digital/ax-code
brew tap defai-digital/ax-code-desktop
brew install ax-code                  # CLI/runtime
brew install --cask ax-code-desktop   # Desktop app
```

If the CLI is already installed, only run the cask command. The Desktop cask token is `ax-code-desktop` —
it intentionally differs from the `ax-code` CLI formula, because Homebrew refuses to link a formula while an
installed cask shares its token (which used to leave the `ax-code` command missing after upgrades). Installs
of the short-lived `ax-code` cask token migrate to `ax-code-desktop` automatically on their next
`brew upgrade`.

You can also download the latest macOS DMG from the [GitHub releases page](https://github.com/defai-digital/ax-code/releases). Open it, drag **AX Code** into **Applications**, and launch it from Applications.

## Windows

Download the latest Windows installer from the [GitHub releases page](https://github.com/defai-digital/ax-code/releases) and run it. For portable use, download the Windows ZIP, extract the full folder, and run `AX Code.exe` from the extracted folder.

## Platform capabilities

| Capability               | macOS Apple Silicon                      | Windows x64   |
| ------------------------ | ---------------------------------------- | ------------- |
| Desktop app              | Supported                                | Supported     |
| Hosted providers         | Supported                                | Supported     |
| AX Engine local provider | Supported on eligible Apple Silicon Macs | Not supported |

AX Engine is a local MLX/Apple Silicon provider. On Windows, use hosted providers, OpenAI-compatible gateways, or connect Desktop to a remote AX Code server running on a supported Mac when AX Engine is required.

## Development checkout

Contributor builds run from the monorepo:

```bash
pnpm install
pnpm run desktop:build
node desktop/packages/electron/scripts/package.mjs --mac zip --publish=never
```
