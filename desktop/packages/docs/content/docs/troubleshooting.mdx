# Troubleshooting

Hit a snag? Find the symptom below and try the fix.

## AX Code Desktop command exits or fails to start

- confirm Node.js `>=20`
- run `ax-code-desktop --version`
- reinstall latest CLI if needed

## Web UI is not reachable

- check server logs with `ax-code-desktop logs`
- verify the active port (preferred default `3100`)
- open the reported `http://localhost:<port>` URL on the same workstation
