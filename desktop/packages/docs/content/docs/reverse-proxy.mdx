# Reverse Proxy

Reverse proxies are unsupported by AX Code Desktop's local-only policy. The server accepts loopback bind addresses only, ignores proxy forwarding headers, and does not trust persisted public origins.

Do not publish AX Code Desktop through Nginx, Caddy, Cloudflare, a VPN relay, or another proxy. Use it directly on the workstation that owns the code. See [Local-only Security](/security/).
