# Remote Instances

Remote host switching and SSH access to AX Code instances are disabled. The Settings tab, onboarding entry points, host switcher, SSH forwarding, and related desktop commands are unavailable in local-only builds.

Git over SSH is unaffected; the restriction applies to remote access to the AX Code service itself. See [Local-only Security](/security/).
