Compare multiple models on the same coding task (arena / best-of-N).

Modes:
- plan (default): fan out structured approach proposals; rank by diversity/risk (advisory, no file writes).
- implement: create a git worktree per contestant, run an implement agent in each, run project verification, rank verify-first. Main workspace is not modified by contestants. Winner is not auto-merged.

Requires modes.arena.enabled=true and at least two connected providers. Prefer verify_first strategy for implement.

Do not use for trivial one-line edits. Do not treat ranking as auto-merge authority.
