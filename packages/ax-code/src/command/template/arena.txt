## Instructions

1. Prefer **≤1 short context gather** (key files/stack). Do not map the whole monorepo before the fan-out.
2. Call **arena** once with the right `mode`. If disabled, call again with `enableIfDisabled: true` (writes `modes.arena.enabled` and continues — no session restart). Do **not** spend many turns debugging config.
3. Arena needs **≥2 connected providers**. If insufficient, stop and tell the user to `/connect` more providers — do not invent rankings.
4. Present ranked results honestly. Never fabricate multi-model output.
5. Never run concurrent writers on the main workspace. Do not auto-merge implement worktrees.
6. If the task is quality findings, switch to **council** instead of plan arena.
