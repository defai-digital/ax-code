# RETIRED — `ax-code-render`

Rust N-API OpenTUI render core (ADR-046). Removed from the Cargo workspace
members list and no longer packaged.

The later "native" successor, `ax-code-tui` (standalone Ratatui UI), was itself
removed in 2026-07; Zig/OpenTUI is the only UI engine. Keep this tree only as
historical reference until a follow-up deletes it entirely.
